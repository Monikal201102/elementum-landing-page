# Elementum Landing Page

A responsive React.js landing page developed from a Figma design as part of an internship assignment.

---

## Project Overview

Elementum is a modern and visually appealing landing page built using React.js and CSS. The project focuses on accurately implementing the provided Figma design while maintaining responsiveness across different screen sizes.

The website consists of multiple sections including:

- Hero Section
- Feature Sections
- Services Section
- Testimonials Section
- Newsletter Subscription CTA
- Footer

---

## Technologies Used

- React.js
- Vite
- JavaScript (ES6+)
- CSS3

## Features

### Hero Section
- Large headline typography
- Decorative vector highlights
- Floating profile images
- Responsive layout

### Feature Section
- Alternating image and text layouts
- Decorative polygons and vectors
- Curved connecting line design

### Services Section
- Service listings with navigation arrows
- Highlighted heading elements
- Decorative confidence badge

### Testimonials Section
- Customer testimonial card
- Decorative quotation marks
- Floating profile images
- Responsive positioning

### Newsletter Subscription CTA
- Large call-to-action heading
- Subscribe button
- Decorative vector and shape elements

### Footer
- Company Links
- Terms & Policies
- Social Media Links
- Contact Information
- Copyright Section

## Installation

Clone the repository:

```bash
git clone <repository-url>
```

Navigate to project directory:

```bash
cd elementum-landing-page
```

Install dependencies:

```bash
npm install
```

Start development server:

```bash
npm run dev
```

---

## Production Build

Create production build:

```bash
npm run build
```

Preview production build:

```bash
npm run preview
```
